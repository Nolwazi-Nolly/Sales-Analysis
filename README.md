# Sales-Analysis
The project focuses on calculating key metrics such as unit price, gross profit, and price elasticity of demand. It also explores sales trends and evaluates how promotional pricing impacts performance and profitability.

**Dataset used:** Sales Case Study.csv

**Tools or technologies used:** 
Microsoft Excel--data cleaning and visualizations
SQL(SSMS)--Quering and analysing sales data
Microsoft Powerpoint--presentation

**Main objective:**
To analyze sales data and generate insights on pricing, profitability, and demand.

**Key insights:**
-Lower prices trigger instant volume spikes
-2015 delivered the biggest loss because costs sat far above price
-High volumes consistently deliver low gross profit
